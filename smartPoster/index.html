SMART POSTER: AN AR EQUIPPED ANDROID APP

WHAT IS SMART POSTER?

Smart Poster is an Android App that redirects users to a third-party website on scanning of NFC tag/QR code from the company product’s advertisement poster. While normal posters usually display an item’s picture, Smart Poster allows users to interact with the product’s 3D AR object that pops up after website redirection to the company’s website. Interaction introduces another dimension that enables users to get acquainted with the product’s specifics. Companies can keep track of number of visits to their website, or number of AR object accesses, through the third-party website.
